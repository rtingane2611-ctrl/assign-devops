# DevOps Assignment Project

## 📌 Description
This project contains a Linux disk monitoring script.

## 🛠 Technologies Used
- Linux (Bash Scripting)
- Git
- GitHub
- GitHub Actions (CI/CD)

## ⚙️ Features
- Checks disk usage
- Alerts if disk > 80%
- Automated execution using CI/CD

## 🚀 CI/CD
GitHub Actions automatically runs the script on every push.

## 📂 Files
- disk_check.sh → Disk monitoring script
